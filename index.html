BASIC HTML AND CSS REFERENCE

HTML TAGS

TAG NAME DESCRIPTION
<html> HTML The root element of an HTML document.
<a> ANCHOR Used to create hyperlink
<li> LIST Defines a list item within <ul> or <ol>
<img> IMAGE Embeds an image into the document
<script> SCRIPT Embeds or links to JavaScript code.

CSS PROPERTIES

PROP DESCRIPTION EXAMPLE
color Sets color of text content color:red
margin Creates space around an element margin: 20px
padding Creates space inside the border padding: 20px
border Defines the appearance of the border border: 1px solid #ccc
display Specifies how an element is displayed display:flex
